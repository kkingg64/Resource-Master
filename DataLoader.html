📊 OMS Data Loader

For Local Testing - Offline Mode

📥 步驟 1: Export Live Data
⏳ 首先係需要export你個live Supabase data到本地JSON files。
⚠️ 注意：呢步需要你個live database credentials係有效嘅
npm run export-data

💡 呢個會生成 data-export/ 資料夾含所有你個data

✅ 步驟 2: Import Data Locally
Export完成後，按下面個button去load data到browser localStorage (offline mode)

或者 → 打開瀏覽器 Console (F12) 執行下面命令

window.__loadDataFromExport()
🔌 步驟 3: Enable Offline Mode
編輯 .env.local 確保有：

VITE_USE_LOCAL_DATA=true
🚀 步驟 4: Start Offline
Refresh browser (F5) 或 restart dev server
